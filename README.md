# Hash_SpeedTest

# How to Build and Run


## Project for Python3

### Change Directory
```
$ cd Project_Python3
```

### Run

```
$ python Main.py ../testdata/data_~~~.txt 4
```

## Project for C Sharp (.NET Core 2.0 or Later)


### Change Directory

```
$ cd Project_CS
```

### Run(dotnet) [Windows/macOS/Linux]

```
> dotnet run ../testdata/data_~~~.txt 4
```

## Project for Java


### Change Directory

```
$ cd Project_Java
```

### Build

* Windows

```
> rm *.class
> mingw32-make all
```

* macOS/Linux

```
$ make clean
$ make all
```


### Run

```
$ java Main ../testdata/data_~~~.txt
```

## Project for Golang


### Change Directory

```
$ cd /Project_Golang
```

### Run

```
$ go run main.go ../testdata/data_~~~.txt 4
```
